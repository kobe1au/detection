#### Bug #2: 小batch对比学习静默失败 📋 待修复
```python
# 问题：batch_size=1时，_info_nce返回0但不报错
if a.size(0) <= 1:
    return a.new_tensor(0.0)  # 静默失败！

# 修复：在训练开始时检查batch_size
if batch_size < 2 and contrast_enabled:
    raise ValueError("需要 batch_size >= 2")
```
**影响**：训练可能在不知情的情况下退化为纯CE损失

#### Bug #3: eval_seed未显式验证 📋 待修复
```python
# 问题：如果config中没有eval.seed，使用默认值2026
eval_seed = int((cfg.get("eval", {}) or {}).get("seed", 2026))

# 修复：强制要求显式设置
if "seed" not in eval_cfg:
    raise ValueError("必须显式设置 eval.seed")
```
**影响**：多seed实验的公平性可能被破坏

### P1级Bug（重要，建议修复）

#### Bug #4: donor索引越界风险
```python
# 当前：越界时静默返回None
donor_idx = self.manifest_donor_indices[idx] if idx < len(...) else None

# 建议：预先验证长度
if len(self.manifest_donor_indices) != len(self.samples):
    raise AEGDatasetConfigError("索引长度不匹配")
```

#### Bug #5: temperature下界硬编码
```python
# 当前：硬编码 1e-4
logits = a @ b.t() / max(temperature, 1e-4)

# 建议：使用模块常量
MIN_TEMPERATURE = 1e-3
logits = a @ b.t() / max(temperature, MIN_TEMPERATURE)
```
